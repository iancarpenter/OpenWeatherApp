# OpenWeatherApp
A Weather app that obtains its data from the [OpenWeatherAPI](https://openweathermap.org/api)

Once running, the browser will ask for location and if successful weather information for the browsers location will be displayed. 

![OpenWeatherApp](https://oraclefrontovikcom.files.wordpress.com/2020/04/openweatherappgithub.png "OpenWeatherApp")

Developed using C#, Angular and Typescript. 

# API Key
An Openweather API key is required which can be obtained from [here](https://openweathermap.org/appid). 

Once obtained it can then be added to this [file](https://github.com/iancarpenter/OpenWeatherApp/blob/master/OpenWeatherAPI.cs)
